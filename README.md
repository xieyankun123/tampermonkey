# 油猴脚本集合

个人使用的 Tampermonkey / Greasemonkey 用户脚本仓库。

## 安装

1. 安装浏览器扩展：[Tampermonkey](https://www.tampermonkey.net/)
2. 打开对应脚本的 `.user.js` 文件，复制全部内容
3. 在 Tampermonkey 中「添加新脚本」，粘贴并保存

或直接点击仓库中的脚本文件，Tampermonkey 会提示安装。
